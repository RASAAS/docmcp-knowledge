# NMPA 医疗器械相关中国标准 / Chinese Medical Device Standards (GB/YY)

本目录收录国家药品监督管理局（NMPA）及国家标准化管理委员会（SAC）发布的医疗器械相关国家标准（GB）和行业标准（YY）。

This directory contains Chinese national standards (GB) and industry standards (YY) for medical devices.

## 说明 / Notes

- **来源**: [SAC 官网](https://www.sac.gov.cn)、[NMPA 官网](https://www.nmpa.gov.cn)
- **范围**:
  - GB 标准：国家强制性或推荐性标准（GB/T、GB/Z）
  - YY 标准：医疗器械行业标准（YY/T）
  - 不包括 ISO/IEC 国际标准（见 `_shared/standards/`）
- **版权说明**: 标准全文受版权保护，本目录仅收录标准编号、标题、适用范围及引用关系

## 官方来源 / Official Sources

- NMPA 标准信息: https://www.nmpa.gov.cn/yljx/ylqxbz/
