# FDA-Recognized Consensus Standards / FDA 认可共识标准

本目录收录 FDA 认可的共识标准，包括 AAMI、ASTM、ANSI、IEEE 等标准组织发布的标准。

This directory contains FDA-recognized consensus standards from AAMI, ASTM, ANSI, IEEE, and other standards development organizations (SDOs).

## 说明 / Notes

- **来源**: [FDA Recognized Consensus Standards Database](https://accessdata.fda.gov/scripts/cdrh/cfdocs/cfStandards/search.cfm)
- **范围**: 仅收录 FDA 正式认可的标准，不包括 ISO/IEC 通用国际标准（见 `_shared/standards/`）
- **主要来源组织**: AAMI, ASTM International, ANSI, IEEE, NEMA
