# Cross-Regulatory Glossary / 跨法规体系术语对照

本目录收录医疗器械合规领域的中英文术语对照，供翻译时参考。

This directory contains Chinese-English terminology for medical device compliance, for reference during translation.

## 文件组织 / File Organization

- `general.json` — 通用术语 / General terms
- `eu_mdr.json` — EU MDR 专有术语 / EU MDR-specific terms
- `fda.json` — FDA 专有术语 / FDA-specific terms
- `nmpa.json` — NMPA 专有术语 / NMPA-specific terms
