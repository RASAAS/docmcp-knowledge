# International Standards / 国际通用标准

本目录收录不专属于单一法规体系的国际通用标准，主要包括 ISO 和 IEC 发布的医疗器械相关标准。

This directory contains international standards not specific to a single regulatory framework, primarily ISO and IEC standards relevant to medical devices.

## 范围区分 / Scope Distinction

| 目录 | 收录范围 |
|------|----------|
| `eu_mdr/standards/` | EU OJ 发布的协调标准（Harmonised Standards） |
| `fda/standards/` | FDA 认可的共识标准（AAMI、ASTM 等） |
| `nmpa/standards/` | 中国 GB/YY 标准 |
| `_shared/standards/`（本目录） | ISO/IEC 国际标准（被多个体系引用） |

## 版权说明 / Copyright Note

ISO/IEC 标准全文受版权保护，本目录**仅收录**：标准编号、标题、适用范围（Scope）、适用的法规体系和引用关系。
