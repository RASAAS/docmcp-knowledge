# MDCG 2020-6：遗留器械充分临床证据指南

**文件编号**：MDCG 2020-6  
**发布日期**：2020年5月  
**适用法规**：EU MDR 2017/745  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/a6d29444-b5d5-4afb-8024-10be85256aa7_en?filename=md_mdcg_2020_6_guidance_sufficient_clinical_evidence_en.pdf)

## 文件目的

本指南阐明了对于已在市场上销售的遗留器械（legacy devices），何为"充分的临床证据"，以满足EU MDR第61条的要求。同时说明了本指南与MEDDEV 2.7/1 rev.4的关系。

## 遗留器械的定义

遗留器械是指在EU MDR适用日期（2021年5月26日）前已根据MDD（93/42/EEC）或AIMDD（90/385/EEC）获得CE标志的器械。

## 充分临床证据的要求

### 核心原则

对于遗留器械，制造商须证明：
1. 器械在其预期目的下具有可接受的收益/风险比
2. 临床证据足以支持CE标志的维持
3. 临床评价符合MDR附件XIV的要求

### 临床证据的来源

**已有临床数据**（优先使用）：
- 制造商自身器械的上市后数据（投诉、不良事件、PMCF数据）
- 已发表的科学文献（关于该器械或等同器械）
- 临床调查数据（如有）

**文献综述**：
- 须使用系统性方法进行文献搜索
- 须评估文献质量和相关性
- 须涵盖安全性和性能两个方面

### 充分性评估标准

临床证据被认为"充分"须满足：
- **数量**：足够数量的患者/使用案例
- **质量**：高质量的临床数据（随机对照试验优于病例系列）
- **相关性**：数据与当前预期目的相关
- **时效性**：数据反映当前技术水平（SOTA）

## 与 MEDDEV 2.7/1 rev.4 的关系

本指南补充了MEDDEV 2.7/1 rev.4，主要差异：

| 方面 | MEDDEV 2.7/1 rev.4 | MDCG 2020-6 / MDR |
|------|-------------------|-------------------|
| 适用范围 | 所有器械 | 专门针对遗留器械 |
| 临床证据要求 | 一般要求 | 更明确的充分性标准 |
| PMCF要求 | 建议 | 强制（须制定PMCF计划） |
| 公告机构审查 | Class III/植入式 | Class III/植入式（更严格） |

## 遗留器械的临床评价策略

### 推荐方法

1. **评估现有临床数据**：收集并评估所有已有临床数据
2. **识别数据缺口**：确定哪些安全性/性能方面缺乏充分证据
3. **制定PMCF计划**：针对数据缺口制定上市后临床跟踪计划
4. **更新CER**：基于现有数据和PMCF计划撰写临床评价报告

### 常见挑战

- **历史数据质量差**：早期上市的器械可能缺乏系统性临床数据
- **等同性证明困难**：MDR对等同性要求更严格
- **PMCF资源需求**：需要投入资源开展PMCF活动
