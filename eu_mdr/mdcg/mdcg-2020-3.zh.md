# MDCG 2020-3 rev.1：过渡期重大变更指南

**文件编号**：MDCG 2020-3 rev.1  
**发布日期**：2023年9月（rev.1）  
**适用法规**：EU MDR 2017/745  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/800e8e87-d4eb-4cc5-b5ad-07a9146d7c90_en?filename=mdcg_2020-3_en_1.pdf)

## 文件目的

本指南阐明了EU MDR第120条过渡期条款中"重大变更"（significant change）的含义，帮助制造商判断遗留器械的变更是否触发须立即按MDR进行合规评估的要求。

## 重大变更的判断标准

### 定义

"重大变更"是指影响器械合规性或可能影响器械安全性和性能的变更。

### 不构成重大变更的情形

以下变更通常**不构成**重大变更：
- 纯粹的行政变更（如制造商地址变更）
- 不影响安全性和性能的轻微设计改进
- 为提高可靠性而进行的小幅改进（不改变预期目的）
- 软件的安全补丁（不增加新功能）

### 构成重大变更的情形

以下变更通常**构成**重大变更，须按MDR进行合规评估：

**预期目的变更**：
- 新增适应症
- 扩展目标患者群体
- 新增使用环境

**设计变更**：
- 材料变更（特别是与人体接触的材料）
- 能量传递方式变更
- 影响安全性的结构变更

**制造工艺变更**：
- 灭菌方法变更
- 关键制造步骤变更

**软件变更**：
- 新增临床功能
- 算法重大更新（影响临床输出）

## 变更评估流程

```
识别变更
    ↓
评估变更是否影响安全性或性能
    ↓
    是 → 重大变更 → 须按MDR进行合规评估
    否 → 非重大变更 → 可继续使用MDD/AIMDD证书（过渡期内）
```

## 文件化要求

制造商须记录变更评估：
- 变更描述
- 评估方法
- 结论（重大/非重大）
- 若为非重大变更：说明理由
