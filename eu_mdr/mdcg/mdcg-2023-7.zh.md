# MDCG 2023-7：临床调查豁免及等同性数据访问指南

**文件编号**：MDCG 2023-7  
**发布日期**：2023年6月  
**适用法规**：EU MDR 2017/745  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/1b5f9cc0-cea0-4459-921f-eaf4b4f80983_en?filename=mdcg_2023-7_en.pdf)

## 文件目的

本指南阐明了EU MDR第61(4)-(6)条关于临床调查豁免的条件，以及第61(5)条关于"充分访问"等同器械数据的要求。

## 临床调查豁免条件

### 第61(4)条 — 一般豁免

以下情形可豁免临床调查：
- 器械是对已上市器械的改进，且改进不影响安全性和性能
- 可证明与已上市等同器械的等同性（须满足第61(5)条）

### 第61(5)条 — 等同性豁免的数据访问要求

**对于 Class III 和植入式器械**：

若制造商欲通过等同性豁免临床调查，须：
1. 与等同器械制造商签订**合同**，明确允许访问技术文件
2. 合同须允许访问足以证明等同性的数据
3. 合同须允许公告机构访问相关数据

**"充分访问"的定义**：

| 数据类型 | 要求 |
|---------|------|
| 技术文件 | 须能访问等同器械的完整技术文件 |
| 临床数据 | 须能访问等同器械的临床评价报告（CER） |
| PMS数据 | 须能访问等同器械的上市后监督数据 |
| 设计文件 | 须能访问足以证明技术等同性的设计文件 |

### 第61(6)条 — 植入式III类器械的特殊要求

植入式III类器械和III类有源器械通常须进行临床调查，除非：
- 使用已上市器械的现有临床数据（须证明等同性，且满足第61(5)条）
- 器械是对已上市器械的改进，且改进不影响安全性和性能

## 实践指导

### 合同要求

等同性合同须明确：
- 允许访问的数据类型和范围
- 访问方式（直接访问、摘要访问等）
- 保密条款
- 公告机构访问权
- 合同有效期

### 公告机构的角色

公告机构须：
- 验证合同的存在和有效性
- 确认合同允许访问足够的数据
- 评估等同性证明的充分性

### 常见问题

**问：若等同器械制造商拒绝签订合同怎么办？**  
答：若无法获得合同，则不能通过等同性豁免临床调查，须进行临床调查。

**问：是否可以使用公开发表的文献替代合同访问？**  
答：公开文献可作为补充证据，但通常不足以替代对技术文件的直接访问（特别是对于Class III器械）。

**问：若等同器械已停产，是否仍需合同？**  
答：若等同器械制造商仍存在，须尝试签订合同。若制造商已不存在，须评估是否有其他方式获得充分数据。
