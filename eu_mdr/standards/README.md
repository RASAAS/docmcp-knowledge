# EU MDR Harmonised Standards / EU MDR 协调标准

本目录收录欧盟官方公报（Official Journal, OJ）发布的 EU MDR/IVDR 协调标准列表。

This directory contains harmonised standards published in the EU Official Journal (OJ) for EU MDR/IVDR.

## 说明 / Notes

- **来源**: EU 官方公报 Commission Implementing Decisions on harmonised standards
- **范围**: 仅收录 OJ 正式发布的协调标准，不包括 ISO/IEC 通用国际标准（见 `_shared/standards/`）
- **格式**: 每条标准记录包含标准编号、标题、OJ 引用、生效日期

## 官方来源 / Official Source

https://ec.europa.eu/health/medical-devices/new-regulations/implementation/standards_en
