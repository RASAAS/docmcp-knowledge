# 其他适用标准（非协调标准）

除协调标准外，医疗器械制造商通常还应用以下国际标准（ISO、IEC、ASTM 等）来证明符合 EU MDR GSPR 要求。虽然这些标准不产生*符合性推定*，但它们代表了公认的技术水平。

::: info 共 **384** 条标准，分布于 **63** 个类别
:::
