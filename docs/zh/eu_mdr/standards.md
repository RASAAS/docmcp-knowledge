# EU MDR 协调标准

协调标准（Harmonised Standards）由欧盟委员会在**欧盟官方公报（OJ）**中发布。适用协调标准可推定符合 EU MDR 对应 GSPR 要求（**合规推定**）。

::: warning 权威来源
协调标准列表由 **Commission Implementing Decision (EU) 2021/1182** 及其修正案维护。请以最新 OJ 发布版本为准。
- [CID (EU) 2021/1182 — EUR-Lex](https://eur-lex.europa.eu/eli/dec_impl/2021/1182/oj)
- [CID (EU) 2024/815 修正案 — EUR-Lex](https://eur-lex.europa.eu/eli/dec_impl/2024/815/oj)
- [欧盟委员会汇总页面](https://health.ec.europa.eu/medical-devices-topics-interest/harmonised-standards_en)
:::
