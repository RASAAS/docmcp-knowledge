# 欧盟医疗器械法规（EU MDR 2017/745）

EU MDR 2017/745 于 **2021年5月26日** 正式适用，取代了 MDD 93/42/EEC 和 AIMDD 90/385/EEC。

## 结构

- **[法规文本](./regulations)** — 完整法规及修正案
- **[协调标准](./standards)** — 欧盟官方公报（OJ）列出的协调标准
- **[MDCG 指南](./mdcg)** — 医疗器械协调小组指南文件
- **[TEAM-NB 立场文件](./team_nb)** — 公告机构立场文件
