# TEAM-NB 立场文件

**TEAM-NB**（公告机构团队）是欧洲医疗器械公告机构协会，其立场文件从公告机构角度提供 MDR/IVDR 实施指导。

| 主题 | 说明 |
|------|------|
| 临床评价 | CER 要求的实践指导 |
| PMCF | 上市后临床跟踪要求 |
| 等效性 | MDR 下的等效性论证 |
