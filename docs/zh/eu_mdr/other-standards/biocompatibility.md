# 生物相容性（非协调标准）

本类别共 **15** 条标准。

::: tip 关于协调状态的说明
本页列出的标准在医疗器械行业广泛使用，但**未列入欧盟官方公报**（Official Journal），不产生EU MDR下的合规推定效力。正式协调的生物相容性标准（ISO 10993第4、9、10、12、15、17、18、23部分，共8条）请参见[协调标准 - 生物相容性](/zh/eu_mdr/standards/biocompatibility)。
:::
