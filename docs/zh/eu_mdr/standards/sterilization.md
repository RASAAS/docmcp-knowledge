# 协调标准 — 灭菌与包装

**官方来源**：[EC Health — Harmonised Standards](https://health.ec.europa.eu/medical-devices-topics-interest/harmonised-standards_en) | 基于 CID (EU) 2021/1182 及修正案 CID 2024/815

## 标准列表

| 标准号 | 灭菌方法 | 状态 |
|--------|---------|------|
| **EN ISO 11135:2014+A1:2019** | 环氧乙烷（EO）灭菌 | 现行有效 |
| **EN ISO 11137-1:2015+A2:2019** | 辐射灭菌 — 第1部分：开发、确认和常规控制 | 现行有效 |
| **EN ISO 11137-2:2015** | 辐射灭菌 — 第2部分：灭菌剂量确定 | 现行有效 |
| **EN ISO 11607-1:2020** | 无菌屏障系统包装 — 第1部分：材料和系统要求 | 现行有效 |
| **EN ISO 11607-2:2020** | 无菌屏障系统包装 — 第2部分：成型、密封和装配过程确认 | 现行有效 |
| **EN ISO 17665-1:2006+A1:2013** | 湿热（蒸汽）灭菌 | 现行有效 |

## 灭菌方法选择

| 灭菌方法 | 适用器械类型 | 关键标准 |
|----------|------------|---------|
| 环氧乙烷（EO） | 热敏感器械、复杂结构器械 | EN ISO 11135 |
| 辐射（γ射线/电子束） | 一次性器械、大批量生产 | EN ISO 11137-1, -2 |
| 湿热（蒸汽） | 耐热耐湿器械、手术器械 | EN ISO 17665-1 |
