# MDCG 2020-13：临床评价评估报告模板（CEAR）

**文件编号**：MDCG 2020-13  
**发布日期**：2020年7月  
**适用法规**：EU MDR 2017/745  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/02f50abc-91db-4ad9-b137-6ffedb690716_en?filename=md_2020-13-cea-report-template_en.pdf)

## 文件目的

本文件提供了公告机构用于评估制造商临床评价报告（CER）的标准模板——临床评价评估报告（CEAR）。

## CEAR的法规背景

根据EU MDR第52条，公告机构在进行合规评估时须审查制造商的临床评价。CEAR是公告机构记录其临床评价审查结论的标准文件。

## CEAR的使用者

CEAR主要由**公告机构**使用，但制造商了解CEAR的内容有助于：
- 理解公告机构的审查重点
- 准备更完整的CER
- 预判公告机构可能提出的问题

## CEAR内容结构

### 第1节 — 器械和制造商信息

- 器械标识（名称、型号、UDI）
- 制造商信息
- 证书信息（证书编号、有效期）

### 第2节 — CER评估摘要

**CER版本信息**：
- CER版本号和日期
- CER作者资质评估

**临床评价范围**：
- 预期目的是否明确定义
- 目标患者群体是否明确
- 适应症和禁忌症是否完整

### 第3节 — 临床证据评估

**文献综述评估**：
- 搜索策略是否系统性
- 纳入/排除标准是否合理
- 文献质量评估是否充分

**等同性评估**（如适用）：
- 等同性证明是否充分
- 三个维度（技术、生物、临床）是否均已评估
- Class III/植入式器械的合同访问是否已建立

**临床数据质量**：
- 数据量是否足够
- 数据质量是否可接受
- 数据相关性是否充分

### 第4节 — 收益/风险评估

- 收益是否明确量化
- 风险是否完整识别
- 收益/风险比是否可接受

### 第5节 — PMCF评估

- PMCF计划是否充分
- PMCF活动是否针对数据缺口
- PMCF时间表是否合理

### 第6节 — 结论

- 临床评价是否符合MDR要求
- 是否需要额外临床数据
- 是否需要修改PMCF计划
- 总体结论（通过/不通过/需补充）

## 对制造商的启示

了解CEAR模板有助于制造商在撰写CER时：
1. 确保CER结构完整，涵盖CEAR所有评估点
2. 提供充分的等同性证明文件
3. 制定针对性的PMCF计划
4. 清晰呈现收益/风险分析
