# MDCG 2019-15 rev.1：I类器械制造商指南

**文件编号**：MDCG 2019-15 rev.1  
**发布日期**：2019年12月  
**适用法规**：EU MDR 2017/745  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/349a2d4c-ea2a-4279-861c-7c063bc077e4_en?filename=md_guidance-manufacturers_en.pdf)

## 文件目的

本指南为I类医疗器械制造商提供实践指导，帮助其理解和满足EU MDR的要求，特别是自我声明合规评估的程序。

## I类器械的合规评估路径

### 自我声明（无需公告机构）

大多数I类器械可通过制造商自我声明进行合规评估：
- 制造商自行评估器械符合GSPR
- 起草EU符合性声明
- 在器械上粘贴CE标志

### 须涉及公告机构的I类器械

以下I类器械须涉及公告机构：
- **I类无菌器械（Is）**：公告机构审查灭菌方面
- **I类测量功能器械（Im）**：公告机构审查测量方面
- **I类可重复使用外科器械（Ir）**：公告机构审查再处理方面

## I类器械的主要义务

### 技术文件

I类器械须建立技术文件，包含：
- 器械描述和规格
- 设计和制造信息
- GSPR符合性证明
- 风险管理文件
- 临床评价报告（CER）
- 标签和说明书

### 质量管理

I类器械制造商须建立质量管理体系（可简化版本），涵盖：
- 设计控制
- 生产控制
- 投诉处理
- 纠正和预防措施

### 上市后监督

I类器械须建立PMS系统，包含：
- 投诉监测
- 不良事件监测
- 文献综述（定期）
- PMS报告（按需更新，无固定频率）

### 临床评价

I类器械须进行临床评价，但通常可通过：
- 文献综述（等同器械数据）
- 已有临床数据分析
- 无需临床调查（除非有特殊需求）

## 常见I类器械示例

| 器械类型 | 分类 | 说明 |
|---------|------|------|
| 非无菌手术器械 | I类 | 如手术刀、镊子 |
| 绷带（非无菌） | I类 | 普通伤口敷料 |
| 助行器 | I类 | 拐杖、助行架 |
| 眼镜架 | I类 | 不含镜片 |
| 无菌手术器械 | Is类 | 须公告机构参与 |
