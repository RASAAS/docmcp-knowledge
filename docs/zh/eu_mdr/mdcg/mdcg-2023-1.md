# MDCG 2023-1：机构自制器械豁免指南（第5(5)条）

**文件编号**：MDCG 2023-1  
**发布日期**：2023年1月  
**适用法规**：EU MDR 2017/745 / EU IVDR 2017/746  
**官方PDF**：[下载](https://health.ec.europa.eu/document/download/05b15d55-1bcf-4e17-99c4-15c706325847_en?filename=mdcg_2023-1_en.pdf)

## 文件目的

本指南阐明了EU MDR/IVDR第5(5)条关于医疗机构自制器械豁免的条件和要求。

## 豁免的适用条件

根据第5(5)条，医疗机构自制器械可豁免MDR/IVDR的大部分要求，但须满足以下**所有**条件：

1. **机构内部使用**：器械仅在制造机构内部使用（不得转让给其他机构）
2. **无商业等同品**：市场上没有满足相同需求的CE标志器械
3. **特定需求**：器械针对特定患者群体的特定需求而制造
4. **质量管理**：机构须建立质量管理体系
5. **透明度**：须向主管机构提供信息

## 豁免的限制

以下情形**不适用**豁免：
- 大规模生产（即使在同一机构内）
- 向其他机构转让器械
- 市场上已有满足需求的CE标志器械

## 须满足的要求

即使豁免，机构仍须：
- 建立质量管理体系（参照EN ISO 13485）
- 进行风险管理（参照EN ISO 14971）
- 进行临床评价（简化版）
- 记录器械的设计和制造信息
- 向主管机构提供年度报告（Class III和植入式器械）
