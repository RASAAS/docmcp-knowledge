# ISO/IEC标准 法规速递

> 医疗器械合规领域的最新法规动态、标准更新和指南发布。

### IEC 62304 第2版 -- 医疗器械软件标准20年来最大修订（2026年8月发布）

**2026-05-22** | ISO/IEC标准 | 标准修订 | !!! HIGH

预定2026年8月12日发布。核心变化：适用范围从"医疗器械软件"扩展至"健康软件"（含健康管理应用、远程监控）；三级安全分类替换为两级软件过程严格度级别（I级和II级）；新增 AI/ML 开发生命周期条款（5.1.15）；网络安全作为核心设计控制集成（引用 IEC 81001-5-1）；ISO 13485 和 ISO 14971 移除规范性引用；遗留软件移至资料性附录；新增敏捷开发指南附录 F。预计2028-2029年完成监管采纳。
