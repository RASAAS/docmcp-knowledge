# 医疗器械国际标准

以下 ISO/IEC 标准适用于多个法规体系（EU MDR、FDA、NMPA）。

| 标准 | 标题 | 适用体系 |
|------|------|----------|
| [ISO 14971:2019](https://www.iso.org/standard/72704.html) | 医疗器械 — 风险管理应用 | EU MDR、FDA、NMPA |
| [ISO 24971:2020](https://www.iso.org/standard/74437.html) | ISO 14971 应用指南 | EU MDR、FDA |
