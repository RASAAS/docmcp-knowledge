# 第一类医疗器械产品目录

为贯彻落实《医疗器械监督管理条例》（国务院令第739号）要求，进一步指导第一类医疗器械备案工作，国家药品监督管理局组织修订了《第一类医疗器械产品目录》，现予发布。该目录自2022年1月1日起施行。  
特此公告。
