# 国家药品监督管理局（NMPA）— 医疗器械

**国家药品监督管理局（NMPA）** 是中国医疗器械监管机构，前身为 CFDA（国家食品药品监督管理总局）。

## 器械分类

| 类别 | 风险 | 注册类型 |
|------|------|----------|
| 第一类 | 低 | 备案 |
| 第二类 | 中 | 省级注册 |
| 第三类 | 高 | 国家注册 |

## 结构

- **[法规文本](./regulations)** — 法律法规、规章
- **[GB/YY 标准](./standards)** — 国家标准和行业标准
- **[审查指导原则](./guidance)** — 按器械类型的注册审查指导原则
