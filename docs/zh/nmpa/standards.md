# 中国医疗器械标准（GB/YY）

中国医疗器械标准由 **SAC**（国家标准化管理委员会）和 **NMPA** 发布：
- **GB**（国家标准）— 强制性国家标准
- **YY**（医药行业标准）— 医疗器械行业标准
- **YY/T** — 推荐性行业标准

::: tip 来源
[全国标准信息公共服务平台](https://openstd.samr.gov.cn)
:::
