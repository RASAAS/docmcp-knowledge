# 中国医疗器械标准（GB / YY）

中国医疗器械标准体系由国家标准和行业标准两部分组成：

| 标准代号 | 类型 | 说明 |
|----------|------|------|
| **GB** | 强制性国家标准 | 由 SAC（国家标准化管理委员会）发布，适用于所有医疗器械 |
| **GB/T** | 推荐性国家标准 | 推荐采用，非强制 |
| **YY** | 强制性行业标准 | 由 NMPA 发布，针对医疗器械行业 |
| **YY/T** | 推荐性行业标准 | 推荐采用，非强制 |

## 标准库概况

截至 2026 年 5 月，我们的标准数据库包含：

| 统计项 | 数量 |
|--------|------|
| 标准总数 | **2,100** |
| 其中 GB 系列 | 334 |
| 其中 YY 系列 | 1,766 |
| 现行有效 | 1,896 |
| 即将实施 | 204 |

标准按技术领域分为两大类：
- **通用技术领域**（228 项）：适用于大多数医疗器械的基础标准
- **专业技术领域**（1,872 项）：特定产品类别的专用标准
