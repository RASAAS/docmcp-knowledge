# Reguverse Hub

欢迎来到 Reguverse Hub -- 法规事务专业人士共同塑造 Reguverse 未来的社区空间。

## 什么是 Reguverse Hub?

Reguverse Hub 是一个开放平台，供 RA 专业人士：

- **提交功能建议** -- 提出并投票支持对您合规工作流最重要的功能创意
- **讨论最佳实践** -- 与同行分享法规情报、技巧和经验
- **跟踪进度** -- 查看哪些功能已规划、正在开发或已完成

## 参与方式

| | 访客 | Reguverse 用户 |
|---|---|---|
| 浏览和阅读 | 可以 | 可以 |
| 提交创意 | 可以（需填写姓名） | 可以（带认证标识） |
| 投票 / 点赞 | 可以（需填写邮箱） | 可以（2倍权重） |
| 评论 | 可以（需填写姓名） | 可以（带认证标识） |

Reguverse 注册用户会获得 **Verified** 认证标识，投票权重为普通用户的2倍。
