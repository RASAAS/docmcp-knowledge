## 关于本知识库

本知识库是一个开放的医疗器械合规知识库，由 [RASAAS](https://github.com/RASAAS) 和社区共同维护。

所有内容均：
- **双语**：提供中文和英文两个版本
- **结构化**：以 Markdown + JSON 格式存储，可被 [DocMCP](https://github.com/RASAAS/docmcp) AI 工具直接消费
- **已验证**：来源 URL 均针对官方机构验证
- **开放**：以 CC BY 4.0 许可证授权
