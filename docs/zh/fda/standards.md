# FDA 认可共识标准

FDA 认可来自 **AAMI**、**ASTM**、**ANSI**、**ISO** 和 **IEC** 的标准。符合认可标准可简化 510(k) 和 PMA 审查。

::: tip
搜索所有 FDA 认可标准：[accessdata.fda.gov](https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfStandards/search.cfm)
:::
