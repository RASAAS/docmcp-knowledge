# FDA 认可共识标准

FDA 认可来自 **AAMI**、**ASTM**、**ANSI**、**ISO** 和 **IEC** 的标准，使用认可标准可简化 510(k) 和 PMA 审查。

## 电气安全
| 标准 | 标题 |
|------|------|
| ANSI/AAMI ES60601-1:2005+A1:2012 | 医用电气设备 — 通用要求 |
| IEC 60601-1-2:2014+AMD1:2020 | 电磁兼容要求 |

## 生物相容性
| 标准 | 标题 |
|------|------|
| ISO 10993-1:2018 | 生物学评价 — 第1部分 |
| ISO 10993-5:2009 | 体外细胞毒性试验 |

## 软件
| 标准 | 标题 |
|------|------|
| IEC 62304:2006+AMD1:2015 | 医疗器械软件生命周期 |
| IEC 82304-1:2016 | 健康软件 — 通用安全要求 |

## 风险管理
| 标准 | 标题 |
|------|------|
| ISO 14971:2019 | 医疗器械风险管理 |
