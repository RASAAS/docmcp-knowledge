# FDA 法规 — 21 CFR

FDA 医疗器械法规编入**联邦法规汇编第 21 卷（21 CFR）**。

## 关键部分

| 部分 | 标题 |
|------|------|
| [21 CFR Part 801](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-801) | 标签 |
| [21 CFR Part 803](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-803) | 医疗器械报告（MDR） |
| [21 CFR Part 806](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-806) | 纠正和召回 |
| [21 CFR Part 807](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-807) | 机构注册和器械上市 |
| [21 CFR Part 812](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-812) | 研究用器械豁免（IDE） |
| [21 CFR Part 814](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-814) | 上市前批准（PMA） |
| [21 CFR Part 820](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-820) | 质量管理体系法规（QMSR） |
| [21 CFR Part 830](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-830) | 唯一器械标识（UDI） |

## 质量体系法规更新（2024）

FDA 于 2024 年更新 **21 CFR Part 820**，与 ISO 13485:2016 对齐，创建**质量管理体系法规（QMSR）**。

- 生效日期：2026年2月2日
- 通过引用纳入 ISO 13485:2016
- [联邦公报通知](https://www.federalregister.gov/documents/2024/02/02/2024-01318/medical-devices-quality-system-regulation-amendments)
