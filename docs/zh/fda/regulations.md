# FDA 法规 — 21 CFR

FDA 医疗器械法规位于**美国联邦法规第21章（21 CFR）**。所有法规：[ecfr.gov/current/title-21](https://www.ecfr.gov/current/title-21)

## 关键部分概览

| 部分 | 标题 | 核心要求 |
|------|------|----------|
| [21 CFR 801](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-801) | 标签 | 充分的使用说明，处方器械标签 |
| [21 CFR 803](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-803) | 医疗器械报告（MDR）| 死亡、严重伤害、故障强制报告 |
| [21 CFR 806](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-806) | 纠正与召回 | 召回和现场纠正报告 |
| [21 CFR 807](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-807) | 机构注册与器械列名 | 510(k) 上市前通知 |
| [21 CFR 812](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-812) | 研究性器械豁免（IDE）| 未批准器械临床研究 |
| [21 CFR 814](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-814) | 上市前批准（PMA）| Class III 器械批准 |
| [21 CFR 820](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-820) | 质量管理体系法规（QMSR）| 与 ISO 13485:2016 对齐的 QMS 要求 |
| [21 CFR 830](https://www.ecfr.gov/current/title-21/chapter-I/subchapter-H/part-830) | 唯一器械标识（UDI）| UDI 标签和 GUDID 数据库 |

## 21 CFR Part 820 — QMSR 关键要求（2026年2月2日生效）

FDA 于2024年更新 Part 820，与 **ISO 13485:2016** 对齐，创建**质量管理体系法规（QMSR）**。

- [联邦公报通知](https://www.federalregister.gov/documents/2024/02/02/2024-01318/medical-devices-quality-system-regulation-amendments)
- 以引用方式纳入 ISO 13485:2016
- 取代原质量体系法规（QSR）

**QMSR 关键子部分：**

| 子部分 | 主题 |
|--------|------|
| 子部分 A | 通用规定（范围、定义）|
| 子部分 B | 质量管理体系（QMS 策划、文件）|
| 子部分 C | 管理职责 |
| 子部分 D | 资源管理 |
| 子部分 E | 产品和服务实现（设计控制、采购）|
| 子部分 F | 测量、分析和改进 |

| 途径 | 法规 | 器械类别 | 说明 |

| IDE | 21 CFR 812 | 研究性 | 临床研究豁免 |
