# 解读FDA GUDID包装配置管理要求：医疗器械UDI合规的关键要素

> Original: https://www.fda.gov/regulatory-information/search-fda-guidance-documents/global-unique-device-identification-database-gudid 编者按： 2024年12月17日，FDA发布了一份关于GUDID包装配置管理的重要指导文件，系统阐述了医疗器械唯一标识系统(UDI)

Original: <https://www.fda.gov/regulatory-information/search-fda-guidance-documents/global-unique-device-identification-database-gudid>

编者按： 

2024年12月17日，FDA发布了一份关于GUDID包装配置管理的重要指导文件，系统阐述了医疗器械唯一标识系统(UDI)在包装层级管理方面的具体要求。

文件通过详实的案例展示了GUDID系统中不同包装层级的配置要求和数据结构。特别值得注意的是，文件对包装层级的继承关系、商业分销状态等关键要素进行了明确定义，这为企业准确提交GUDID数据提供了清晰指引。

GUDID包装配置管理呈现以下特点：

第一，层级化的标识管理策略：

  * 基础包装层级需要有原始DI标识
  * 不同包装层级各自配备独立的包装DI
  * 包装标识之间存在明确的层级关系和数量对应关系

第二，属性继承机制：

  * 上层包装自动继承基础包装的属性信息
  * 特定属性（如数量、包装类型等）可在各层级独立定义
  * 系统自动管理属性继承关系，减少数据重复录入

第三，灵活的包装配置支持：

  * 支持单层次包装配置（如盒装100支注射器）
  * 支持多层次包装配置（如导管的盒装和箱装）
  * 允许同一基础产品有多种包装配置方案

第四，严格的状态管理要求：

  * 包装状态需要明确标识（商业分销/非商业分销）
  * 通过停用日期自动管理包装状态
  * 确保市场流通产品的可追溯性

值得强调的是，这份指导文件展现了FDA在医疗器械UDI管理领域的系统性思维。通过建立标准化的包装配置管理体系，既确保了产品在供应链中的可追溯性，又为制造商提供了灵活的包装方案选择空间。

【编后语】 对中国医疗器械企业而言，理解并正确实施GUDID包装配置管理至关重要。建议企业应当： （1）建立完整的包装层级管理体系，确保DI编码的准确性和唯一性； （2）开发配套的信息管理系统，支持多层级包装数据的维护； （3）建立包装配置变更管理机制，确保GUDID数据及时更新； （4）加强与FDA认可发证机构的沟通，确保DI编码符合要求。

只有建立起完善的GUDID包装配置管理体系，企业才能在美国市场建立起可靠的产品追溯体系，从而在这一全球最大的医疗器械市场中保持持续竞争力。
