# 无菌和化学性能检验样品数量

> 无菌测试 按《中国药典》2020版四部通则1101无菌检查法中规定的直接接种法，使用金黄色葡萄球菌进行检测。 样品准备至少三套。按型号准备，不考虑每套中的产品数量。 化学性能 取样品，按0.2g样品加1ml的比例加水，37℃±1℃条件下，浸提24h，将样品与液体分离，冷至室温，作为检验液。 上述项目共计需要浸提液250mL，按配比计算需样品至少50克。

## 无菌测试

按《中国药典》2020版四部通则1101无菌检查法中规定的直接接种法，使用金黄色葡萄球菌进行检测。

样品准备至少三套。按型号准备，不考虑每套中的产品数量。

## 化学性能

  * 还原物质：10mL检验液与空白液相比，高锰酸钾溶液[c(1/5KMnO4)=0.01mol/L]消耗总量应不超过1.5mL。
  * 重金属含量：检验液中钡、铬、铜、铅、锡总金属含量不得超过1μg/mL，镉含量应小于0.1μg/mL。
  * 酸碱度：检验液与空白对照液的PH值之差应不超过1.5。
  * 蒸发残渣：检验液蒸发残渣的总重量应不超过2.0mg。
  * 紫外吸光度：在波长为220-340nm处，浸提液的吸光度应不大于0.1。
  * 环氧乙烷残留：环氧乙烷残留量应不大于10μg/g。

取样品，按0.2g样品加1ml的比例加水，37℃±1℃条件下，浸提24h，将样品与液体分离，冷至室温，作为检验液。

上述项目共计需要浸提液250mL，按配比计算需样品至少50克。
