# Reguverse Hub

Welcome to the Reguverse Hub -- the community space where regulatory affairs professionals shape the future of Reguverse.

## What is Reguverse Hub?

Reguverse Hub is an open platform for RA professionals to:

- **Request Features** -- Submit and vote on feature ideas that matter most to your compliance workflow
- **Discuss Best Practices** -- Share regulatory intelligence, tips, and experiences with peers
- **Track Progress** -- See which features are planned, in progress, or completed

## How It Works

| | Guest | Reguverse User |
|---|---|---|
| Browse & Read | Yes | Yes |
| Submit Ideas | Yes (with name) | Yes (verified badge) |
| Vote / Like | Yes (with email) | Yes (2x weight) |
| Comment | Yes (with name) | Yes (verified badge) |

Reguverse registered users get a **Verified** badge and their votes count double.
