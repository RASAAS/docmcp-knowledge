# NMPA (国家药品监督管理局) — Medical Devices

The **National Medical Products Administration (NMPA)** is China's regulatory authority for medical devices, formerly known as CFDA.

## Device Classification

| Class | Risk | Registration Type |
|-------|------|-------------------|
| Class I | Low | Filing (备案) |
| Class II | Moderate | Provincial registration |
| Class III | High | National registration |

- **[Regulations](./regulations)** — Laws, regulations, administrative measures
