# Biocompatibility

This category contains **9** non-harmonised standards.

::: tip Note
ISO 10993 series parts that have been designated as EU harmonised standards (such as EN ISO 10993-1:2025, EN ISO 10993-4:2017, EN ISO 10993-5:2009, etc.) and the EN ISO 22442 series have been moved to the [Harmonised Standards - Biocompatibility](/en/eu_mdr/standards/biocompatibility) page. This page only lists supplementary ISO standards that have not been granted harmonised standard status.
:::
