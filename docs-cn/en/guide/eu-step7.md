# Full-text Appraisal (Step 7)

Step 7 performs methodological quality assessment of included literature using scoring criteria (Jadad, CASP).

## Scoring Dimensions

| Dimension | Description | Score |
|-----------|-------------|-------|
| Design (D) | Study design quality | 0-3 |
| Applicability (A) | Relevance to target device | 0-3 |
| Population (P) | Study population relevance | 0-3 |
| Reporting (R) | Completeness of reporting | 0-3 |
| **Total** | **SUM = D + A + P + R** | **0-12** |

## Batch Processing

Similar to Step 4: processes literature in batches with auto-continue mode and progress tracking.

## Output

- Detailed scoring table per article
- Final inclusion/exclusion determination
- Quality assessment summary statistics
