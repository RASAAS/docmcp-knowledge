# Translation

AI-powered translation tool designed specifically for medical device regulatory documents.

## Modes

| Mode | Description |
|------|-------------|
| Word In-place | Select text in Word, translate and replace or insert |
| Excel Batch | Upload Excel, batch translate specified columns |

## Word In-place Translation

1. Select text in Word document
2. Open translation tool
3. Choose target language (CN ↔ EN)
4. Click "Translate"
5. Result can replace original, insert below, or copy to clipboard

## Excel Batch Translation

1. Select "Excel Batch" mode
2. Upload Excel file
3. Choose columns to translate
4. Select target language
5. Download translated file

## Terminology

Optimized for: EU MDR/IVDR, NMPA, clinical evaluation, risk management, and post-market surveillance terminology.
