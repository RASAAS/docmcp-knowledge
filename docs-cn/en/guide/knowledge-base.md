# Knowledge Base

## Overview

The regulatory knowledge base provides full-text search across EU MDR regulations and MDCG guidance documents.

**Available on:** All plans

## Content

| Document | Description |
|----------|-------------|
| EU MDR (Regulation 2017/745) | Full regulation text |
| EU MDR Annex I-XVII | All annexes |
| MDCG Guidance Documents | Medical Device Coordination Group guidelines |

## Features

### Full-text Search

- Keyword search across all documents
- Direct article number navigation
- Highlighted search results
- English and Chinese search support

### Workflow Integration

During clinical evaluation or GSPR compliance checks, relevant regulatory articles are automatically referenced by the AI.

## Access

Click the "Regulations" tab in the plugin to open the knowledge base browser.
