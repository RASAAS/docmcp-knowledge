## Overview

Reguverse Assistant is an AI-powered medical device regulatory intelligence platform delivered as a Microsoft Word Add-in. It supports clinical evaluation and registration dossier generation workflows under EU MDR, FDA, NMPA, and other regulatory frameworks.

## Basic Workflow

1. **Download & Install** -- Download the installer from the official website and install the Word plugin
2. **Register** -- Create your account in the plugin (CN personal users require identity verification)
3. **Create Project** -- Select the regulatory framework and device information
4. **Step-by-step Execution** -- Follow the workflow guidance to complete each analysis step
5. **Generate Documents** -- After approving all steps, generate compliance documents with one click
