# Account Management

## Overview

Click the "Account" icon in the bottom navigation to manage your profile, subscription, and security settings.

## Subscription Management

### View Current Plan

The account page displays:
- Current plan name and tier
- Monthly credits used / total
- Subscription expiry date

### Upgrade Plan

1. Click "Upgrade"
2. Select target plan and billing cycle (monthly/yearly)
3. Confirm order
4. Complete payment
5. Immediate activation (from trial) or effective at current period end

### Purchase Additional Credits

1. Click "Top Up"
2. Enter purchase quantity
3. Complete payment -- credits are available immediately

## Security Settings

### Two-Factor Authentication

- **Enable 2FA**: Scan QR code with authenticator app
- **Recovery Codes**: Save during setup for account recovery
- **Trusted Devices**: 7-day 2FA bypass per device

### Password Change

Available in security settings section.

## Delete Account

::: danger Warning
Account deletion permanently removes all data (projects, documents, step results). This action is irreversible.
:::
