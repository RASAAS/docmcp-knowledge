# Literature Screening (Step 4)

Step 4 performs title/abstract screening on uploaded literature, determining relevance based on inclusion/exclusion criteria.

## Batch Processing

- AI processes literature in batches
- Auto-continue mode runs batches sequentially
- Progress bar shows real-time status
- Can pause for manual batch-by-batch review

## Output

Each article receives:
- **Relevant**: Included for full-text appraisal
- **Irrelevant**: Excluded with documented reason

Summary statistics: total retrieved, after deduplication, included, excluded (by reason).
