# 法规知识库

## 概述

法规知识库提供 EU MDR 法规原文和 MDCG 指导文件的全文检索功能，帮助 RA 从业者快速定位和查阅法规条文。

**适用套餐：** 全部套餐

## 功能

### 法规浏览

在知识库中可浏览以下法规文档：

| 法规 | 说明 |
|------|------|
| EU MDR (Regulation 2017/745) | 欧盟医疗器械法规全文 |
| EU MDR Annex I | 通用安全性能要求 (GSPR) |
| EU MDR Annex II-XVII | 其他附录 |
| MDCG 指导文件 | 医疗器械协调组指南 |

### 全文检索

- 支持关键词搜索
- 支持条款编号直接定位
- 搜索结果高亮显示
- 支持中英文检索

<!-- 截图占位：法规知识库检索界面 -->

### 在工作流中引用

在临床评价或 GSPR 合规检查过程中，可直接从知识库引用相关条款：
- AI 自动关联相关法规条款
- 点击引用可查看完整条文
- 支持在文档中插入法规引用

## 访问方式

在插件中点击「Regulations」标签即可进入法规知识库浏览器。

## 数据更新

法规知识库内容定期更新以反映最新的法规修订和新发布的 MDCG 指导文件。
