# 账户管理

## 账户页面概览

在插件底部导航栏点击「Account」图标进入账户页面，可管理个人信息、订阅和安全设置。

<!-- 截图占位：账户页面总览 -->

## 个人信息

显示当前账户的基本信息：

- 邮箱地址
- 用户类型（个人 / 企业）
- 当前套餐
- 订阅到期时间
- Credits 余额与使用情况

## 订阅管理

### 查看当前订阅

账户页面顶部显示：
- 当前套餐名称与等级
- 本月 Credits 使用量 / 总额度
- 订阅到期日期

### 升级套餐

1. 点击「Upgrade / 升级」按钮
2. 选择目标套餐和付费周期（月付/年付）
3. 确认订单金额
4. 通过支付宝完成付款
5. 升级立即生效（从试用升级时）或在当前周期结束时生效

### 购买额外 Credits

当月额度不足时：
1. 点击「Top Up / 充值」按钮
2. 输入购买数量
3. 完成支付后立即到账

## 安全设置

### 双因素认证 (2FA)

- **启用 2FA**：点击「Enable 2FA」，扫描二维码绑定认证器应用
- **关闭 2FA**：输入当前验证码确认后关闭（仅 Trial 用户可关闭）
- **恢复码**：启用时会生成一组恢复码，务必妥善保存

### 修改密码

在安全设置中可修改登录密码。

### 手机号管理

中国个人用户可在此处查看或更新已验证的手机号。

## 企业认证（中国企业用户）

中国企业用户需完成以下认证流程：

1. **企业信息审核**：提交统一社会信用代码 (USCC)，等待系统验证
2. **合同签约**：审核通过后，在线签署服务协议
3. **付款激活**：签约完成后即可选择套餐并付款

认证状态会在账户页面实时显示：
- 待审核 → 已通过 → 已签约 → 已激活

## 管理员功能

管理员和超级管理员可在账户页面下方看到「Admin Panel」入口，提供：

- 用户管理（查看/修改用户状态）
- 邀请码管理
- 订阅请求处理
- AI 工具管理
- 合同审核
- 系统通知发送
- 功能可见性控制

## 删除账户

如需删除账户，在账户设置最底部点击「Delete Account / 删除账户」。

::: danger 警告
删除账户将永久清除您的所有数据（项目、文档、步骤结果等），此操作不可逆。
:::
