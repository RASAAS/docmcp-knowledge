# 设备信息 (Step 1)

## 概述

Step 1 是临床评价工作流的起点，用于确认器械的预期用途 (Intended Purpose)、适应症和目标人群。这些信息将贯穿后续所有步骤。

## 操作流程

1. **自动生成**：AI 根据项目设备信息和上下文文件，自动生成预期用途描述
2. **审阅内容**：检查生成的内容是否准确完整
3. **编辑修改**：如有不准确之处可直接编辑
4. **批准确认**：确认无误后点击「Approve / 批准」

## 生成内容包括

- 器械名称与分类
- 预期用途描述
- 适应症与禁忌症
- 目标患者人群
- 预期使用者
- 使用环境与条件
- 作用机理
- 预期临床效果

## 注意事项

- 预期用途是 CEP/CER 文档的核心基础，请仔细审阅
- 内容应与器械 IFU (说明书) 中的表述一致
- 如需修改，可点击「Edit / 编辑」进入编辑模式
