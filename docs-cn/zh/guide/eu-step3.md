# 文献检索 (Step 3)

## 概述

Step 3 定义系统性文献检索策略，包括数据库选择、检索词、纳入/排除标准等。

## 操作流程

1. **自动生成**：AI 生成系统性的检索策略
2. **审阅**：检查检索词和标准是否合理
3. **上传 RIS 文件**：从 PubMed、Embase 等数据库导出 RIS 格式的检索结果
4. **批准**：确认检索策略后进入筛选阶段

## 生成内容包括

- 检索数据库列表 (PubMed, Embase, Cochrane 等)
- 检索词策略 (含 MeSH terms)
- 纳入标准 (Inclusion criteria)
- 排除标准 (Exclusion criteria)
- 检索时间范围
- 文献类型限定

## 上传检索结果

执行检索后，需上传各数据库导出的 RIS 文件：

1. 在 Step 3 面板中点击「Upload RIS / 上传 RIS」
2. 选择对应数据库（PubMed / Embase / Cochrane 等）
3. 上传 `.ris` 格式文件
4. 系统自动统计文献数量和去重结果

::: tip 支持的数据库
PubMed, Embase, Cochrane Library, Web of Science, CNKI (中国知网)
:::
