# 证据注册表 (Evidence Registry)

## 概述

证据注册表 (Evidence Registry) 用于管理和注册风险控制措施的验证证据，建立控制措施到证据来源的追溯链。

::: info 当前状态
目前证据注册表主要服务于风险管理工作流中风险控制措施的验证证据注册。该功能仍在持续优化中。
:::

**适用套餐：** Pro / Max / CRO

## 功能

### 证据登记

- 为每条风险控制措施关联验证证据
- 支持多种证据类型：测试报告、标准符合性声明、临床数据、文献等
- 记录证据来源、日期和有效性

### 追溯管理

- 建立控制措施 → 证据的对应关系
- 支持一条控制措施关联多条证据
- 追踪证据的有效状态

### 与风险管理集成

- 在风险管理面板中可直接为控制措施添加证据
- 证据状态反映在 RCM 矩阵中
