# 文献综述 (Step 8)

## 概述

Step 8 对所有通过全文评审的文献进行综合分析，生成文献数据综述，总结安全性和性能的临床证据。

## 操作流程

1. **自动生成**：AI 综合所有通过评审的文献生成综述
2. **审阅**：检查综述内容的全面性和准确性
3. **编辑**：补充遗漏或修正不准确之处
4. **批准**：确认文献综述

## 生成内容

- 文献数据汇总表
- 临床安全性证据综述
- 临床性能证据综述
- 证据质量总体评估
- 临床数据充分性分析
- 与预期用途的关联性分析
