# 注册与登录

## 注册账号

打开 Reguverse 助手面板后，点击「注册」按钮进入注册流程。

### 注册步骤

注册流程会根据您的地区和用户类型有所不同：

#### 中国个人用户

1. **选择地区**：选择「中国大陆」
2. **选择用户类型**：选择「个人」
3. **手机验证**：输入手机号并完成短信验证码验证
4. **填写信息**：输入邮箱、设置密码
5. **邮箱验证**：输入邮箱收到的验证码完成注册
6. **同意条款**：阅读并同意服务条款和隐私政策

#### 中国企业用户

1. **选择地区**：选择「中国大陆」
2. **选择用户类型**：选择「企业」
3. **填写信息**：输入邮箱、公司名称、设置密码
4. **邮箱验证**：完成邮箱验证码验证
5. **同意条款**：阅读并同意服务条款和隐私政策

::: info 企业用户认证
企业用户注册后需完成企业认证（提交统一社会信用代码等）和合同签约流程后，方可使用付费功能。
:::

#### 国际用户

1. **选择地区**：选择「International / 其他地区」
2. **选择用户类型**：选择「Personal」或「Enterprise」
3. **填写信息**：输入邮箱和密码
4. **邮箱验证**：完成验证码验证
5. **同意条款**：阅读并同意 Terms of Service 和 Privacy Policy

## 登录

### 常规登录

1. 输入注册邮箱和密码
2. 点击「登录」

### 双因素认证 (2FA)

所有付费套餐用户必须启用 TOTP 双因素认证：

1. 首次升级付费套餐后，系统提示设置 2FA
2. 使用认证器应用 (如 Google Authenticator) 扫描二维码
3. 输入 6 位验证码确认绑定
4. **务必保存恢复码**（用于认证器丢失时找回账号）

之后每次登录需输入认证器中的 6 位动态码。

::: tip 信任设备
验证 2FA 时可勾选「信任此设备 7 天」，7 天内无需重复输入验证码。
:::

## 忘记密码

1. 在登录页面点击「忘记密码」
2. 输入注册邮箱
3. 检查邮箱中的重置链接
4. 设置新密码

## 账号安全建议

- 使用强密码（至少 8 位，包含字母和数字）
- 及时启用双因素认证
- 妥善保管 2FA 恢复码
- 不要将账号密码分享给他人
