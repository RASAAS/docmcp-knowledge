# 创建 NMPA 项目

::: info 适用套餐
Pro / Max / CRO
:::

## 创建步骤

### 1. 选择法规体系

在创建项目对话框中，法规体系选择「NMPA」。

### 2. 填写设备信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 项目名称 | 项目描述性名称 | 是 |
| 设备中文名称 | NMPA 注册的正式中文名称 | 是 |
| 设备英文名称 | 设备英文名称 | 建议 |
| 分类编号 | NMPA 分类目录编号 | 是 |
| 注册类型 | 进口 / 国产 | 是 |
| 制造商 | 制造商名称 | 是 |

### 3. 确认创建

点击创建后进入项目仪表板。

## 添加任务

在项目仪表板中可添加：

- **NMPA Registration**：完整注册资料生成
- **Clinical Evaluation (NMPA)**：NMPA 要求的临床评价
- **CE Exempt**：临床评价豁免说明

## 上下文文件

建议上传以下参考文件供 AI 使用：
- 产品技术说明书
- 设计和制造信息
- 检验报告
- 境外上市证明（进口器械）
- 既有的注册资料
