# 临床背景 (Step 2)

## 概述

Step 2 生成器械相关的临床背景信息，包括疾病背景、现有治疗方案、技术现状等，为后续的文献检索和安全性分析提供上下文。

## 操作流程

1. **自动生成**：AI 基于 Step 1 的预期用途，分析相关临床领域背景
2. **审阅内容**：检查临床背景描述的准确性和完整性
3. **编辑修改**：补充或修正内容
4. **批准确认**：点击「Approve」进入下一步

## 生成内容包括

- 相关疾病/病症概述
- 流行病学数据
- 现有治疗方案与替代技术
- 技术发展现状
- 临床需求分析
- 器械在诊疗路径中的定位
