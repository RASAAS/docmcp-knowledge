# AI 工具总览

Reguverse 内置多款 AI 辅助工具，帮助 RA 从业者完成日常法规工作。

在插件底部导航栏点击「AI Tools」图标进入工具页面。

## 可用工具

### MDR/IVDR 设备分类助手

根据 EU MDR Annex VIII 或 EU IVDR Annex VIII 的分类规则，辅助判定医疗器械的分类等级。

**功能特点：**
- 支持 MDR (Class I / IIa / IIb / III) 和 IVDR (A / B / C / D) 分类
- 基于 MDCG 2021-24 指导文件的规则验证
- 提供分类依据和引用的法规条款
- 输出可直接用于技术文档

**使用方式：**
1. 描述器械的预期用途和技术特征
2. AI 分析适用的分类规则
3. 输出分类结果和规则引用

**适用套餐：** 全部套餐

---

### 等同器械研究

利用 FDA 510(k) 数据库和网络检索，帮助识别和评估等同器械 (equivalent device)。

**功能特点：**
- 自动检索 FDA 510(k) 数据库
- 分析临床、技术、生物学等同性
- 输出结构化的等同性论证
- 可直接引用到 CER 文档

**使用方式：**
1. 输入目标器械的基本信息
2. AI 自动检索候选等同器械
3. 对比分析各维度的等同性
4. 生成等同器械论证报告

**适用套餐：** 全部套餐

---

### 文档翻译

专为医疗器械法规文档设计的 AI 翻译工具，支持中英双语互译。

**功能特点：**
- 医疗器械法规术语准确翻译
- 支持 Word 文档原位翻译（保持格式）
- 支持 Excel 文件批量翻译
- 支持选中文本直接翻译

**使用方式：**
- **Word 原位翻译**：选中文本后点击翻译，结果直接替换原文或插入下方
- **文件上传翻译**：上传 Excel 文件，批量翻译指定列

**适用套餐：** 全部套餐

→ [翻译工具详细说明](./translation)

---

### EUDAMED UDI 批量上传

为拥有大量 UDI-DI 编码的制造商提供批量生成 EUDAMED XML 的能力。

**功能特点：**
- 支持 Excel 数据导入（用户自有格式 + 标准模板）
- AI 辅助解析和字段映射
- 生成符合 EUDAMED XSD v3.0.28 规范的 XML
- 自动拆分为多文件（每文件 40 个标识符上限）
- 打包 ZIP 下载

**适用套餐：** 全部套餐

→ [EUDAMED UDI 详细说明](./eudamed-udi)

## 工具使用方式

所有 AI 工具的通用操作流程：

1. 在 AI Tools 页面点击目标工具
2. 按界面提示输入信息或上传文件
3. AI 自动处理并生成结果
4. 审阅结果后可：
   - 插入到 Word 文档
   - 复制到剪贴板
   - 下载文件

## 套餐限制

- 全部 AI 工具对所有套餐可用
- 使用 AI 工具会消耗 Credits
- Credits 用完后需充值或等待下月恢复
